# Space Shooter

I developed a Space Shooter using Unity where the player takes the control of a space ship with the objective of going through different waves of enemies with to reach the final boss and stop it from destroying the player's home. The game is available to play on [itch.io](https://adiaz23.itch.io/space-shooter)

![image](https://github.com/user-attachments/assets/14b3c98e-8300-4a43-8f57-2157b0012a7f)

## Mechanics

The game mechanics are:

- Progress throgh differents waves of enemies
- Space ship controlled by the player.
- Two types of enemies.
- A HUD that shows the player's lives.
- Different sound effects for shooting, destroying, and receiving damage.
- Background music.
- A boss fight at the end of the last level.
- A main menu from which the user can access the game or the option menu.
- The player can adjust the volume of the music and SFX.
- A pause screen (only available for desktop).
- A Game Over and Win screen which allows you to go to main menu or restart the game

## Controllers

**Desktop**
The Spaceship can be controlled using either the arrow keys or the WASD keys. To shoot, press the spacebar. To open the Pause menu, press Escape.

**Mobile**
On mobile, a joystick and a button will be available on the screen to play.
